# CVE Reports

This branch holds timestamped beats CVE reports generated by the nightly build workflow.

## File Naming

- \eports/beats-cve-YYYY-MM-DD.json\ — weekly filtered Inspector findings (JSON)
- \eports/beats-cve-YYYY-MM-DD.csv\ — weekly filtered Inspector findings (CSV)
- \eports/beats-cve-latest.json\ — copy of most recent weekly report (JSON)
- \eports/beats-cve-latest.csv\ — copy of most recent weekly report (CSV)
